# Outputs

Generated pipeline outputs are written here.

**Expected files:**
- Annotated `.mp4` videos (e.g. `picklist_105_annotated.mp4`)
- Predicted state `.csv` files (e.g. `predicted_states.csv`, `smoke_two_stage.csv`)
- Subfolders such as `gt_overlay_smoke/` with overlay videos

These are produced by inference and evaluation runs. Contents of this folder are gitignored.
